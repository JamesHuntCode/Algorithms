-Arrays
- Lists
- Linked Lists
- Stacks
- Queues
- Trees
- Graphs

Algorithms Examples Included:

- Linear Search
- Binary Search (Iterative & Recursive)
- Graph & Tree Traversal
- Sorting (Selection, Integer, Merge, Bucket, Quicksort etc)
